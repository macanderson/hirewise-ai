## Getting Started

1. **Prerequisites**
   - Node.js 18+
   - Python 3.13
   - PNPM

2. **Installation**
   ```bash
   # Clone the repository
   git clone https://github.com/yourusername/hirewise.git
   cd hirewise

   # Install dependencies
   pnpm install
   ```

3. **Development**
   ```bash
   # Start the development server
   pnpm dev
   ```

4. **Build**
   ```bash
   # Build all packages and applications
   pnpm build
   ```
